# gong-cli

## what does this package do ?

> basic command

```
Usage: gong-cli [options] [command]

Options:
  -v --version  output the version number
  -h, --help    output usage information

Commands:
  init  <projectName>        gong-cli init
```

- quickly init a react template project

```bash
gong-cli init demo
cd demo
npm install
```

then you got it!
